# How to find out what is going on.

## Top nach CPU - Auslastung 

```
# High to low
top -o +%CPU
```

## Top nach Speicher - Auslastung 

```
top -o +%MEM

```

## Netzwerkauslastung 

```
bmon 
```

## Plattenauslastung 

```
iotop
# Sortierung : 
# Pfeiltaste links und rechts
# R - reverse

```
